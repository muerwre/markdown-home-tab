# Markdown Extension

Markdown Editor for new tab made with [Milkdown](https://milkdown.dev/) and
[Dockview](https://dockview.dev).

## Installation

```
yarn global add web-ext
yarn
yarn package
```

## Development

The live server will be started, so you can see all changes in a realtime.

```
yarn
yarn dev
```
